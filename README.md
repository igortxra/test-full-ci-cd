# test-full-ci-cd
Experimental repository to test an CI/CD flow in github actions

**Contains**

- Basic flask app
- Use of poetry and pyproject.toml
- Makefile to commands abstraction
- Unit tests
- 100% of test coverage
- Quality Pipeline with:
  - Lint
  - Tests
  - Test coverage
  - Security Checks
- Auto generate tags and releases
- Deploy pipeline for:
  - Development
  - Staging
  - Production
